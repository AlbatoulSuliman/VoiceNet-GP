# VoiceNet: Biometric Recognition Using Voice
VoiceNet is a deep learning-based framework designed for robust and non-intrusive biometric identity verification using voice signals.

# How It Works:
Feature Extraction:Transforms raw voice inputs into Spectrograms or MFCCs(Mel-frequency cepstral coefficients) to capture acoustic features.
Deep Learning Model:Architecture utilizes a Convolutional Neural Network (CNN) to learn unique speaker-specific patterns.
Data Augmentation:Applies noise injection, pitch shifting, and time stretching to enhance model generalization and resilience against environmental noise, varying accents, and speaking styles.

# Evaluation Metrics:
The system is trained on public voice datasets and evaluated using:
Accuracy, Precision, and Recall.
EER (Equal Error Rate).
